# Overview
Built with ReactJS, JavaScript, Bootstrap, CSS and HTML. Showcases a dark mode toggle, reusable react components, cards, API, beautiful UI by CSS styling, CSS variables, date and time and easily accessible navigation links.

# Composition
The page was built with React, Javascript, Bootstrap, CSS and HTML. Reusable components, cards, API data, Bootstrap icons and images are used also. A simple CSS animation is also included.

# Functionality
The page showcases my skills and also demonstrates my application of some of those skills. The hyperlinks direct to relevant areas. A user is able to directly place a call, send an SMS, a whatsapp message, or an email to me directly from the webpage by clicking on the appropriate weblink. Some other links are also availbale which link to my other works, while others link to their description. The icons also have hyperlinks that direct to relevant webpages.

# Commands
A user can scroll freely through the page, which is mobile responsive. There is a `Dark mode` toggle button, a `Go to NavBar` a `Go up` as well as numerous hyperlinks. A user is able to directly place a call, send an SMS, Whatsapp message, email etc by clicking on the appropriate hyperlink.
